# Folkara

> Human Resource Management app — available on the App Store

## Overview

Folkara is a production iOS app that streamlines HR workflows for businesses. It handles day-to-day employee operations including attendance tracking, leave management, onboarding flows, and supports teams across multiple languages.

## Features

- Attendance tracking with check-in/check-out
- Leave request and approval workflows
- Employee onboarding experience
- Multi-language support
- Push notifications for approvals and reminders

## Tech Stack

| Area | Technology |
|------|-----------|
| UI | SwiftUI |
| Architecture | MVVM |
| Networking | REST API |
| CI/CD | Fastlane |
| Environments | Multi-env (dev/staging/prod) |
| Notifications | APNs / Push Notifications |

## Highlights

- Shipped to the App Store and used by real teams
- Multi-environment setup with Fastlane automating build, signing, and deployment
- Designed for scalability across different company sizes and languages
