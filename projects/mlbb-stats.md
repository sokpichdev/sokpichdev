# mlbb-stats

> MLBB MetaSight — hero stats, rankings, and counters with a Metal-powered splash screen

## Overview

A Mobile Legends: Bang Bang companion app that surfaces hero statistics, current meta rankings, and counter-pick recommendations. Features a visually rich splash screen rendered with Metal for GPU-accelerated graphics.

## Features

- Hero stats browser with detailed breakdowns
- Meta rankings updated by tier
- Counter-pick suggestions per hero
- Metal-powered animated splash screen
- Targets iOS 16+

## Tech Stack

| Area | Technology |
|------|-----------|
| UI | SwiftUI |
| Concurrency | async/await |
| Graphics | Metal |
| Minimum OS | iOS 16+ |

## Highlights

- Metal splash screen delivers GPU-rendered visuals that wouldn't be possible with standard SwiftUI animations
- async/await throughout keeps data fetching clean and readable
- iOS 16+ baseline allows use of the latest SwiftUI layout APIs without fallback hacks
