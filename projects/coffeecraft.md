# CoffeeCraft

> Production-grade coffee shop app with dual roles and real-time operations

## Overview

CoffeeCraft is a full-featured coffee shop iOS app built to simulate a real-world ordering system. It supports two distinct user roles — customer and admin — with real-time order tracking, an in-app wallet, and a live admin dashboard.

## Features

- Customer role: browse menu, place orders, track status in real time
- Admin role: manage orders, update menu, view dashboard
- In-app wallet with balance top-up and payment flow
- Real-time order updates
- Location integration with MapKit
- Firebase backend for auth, database, and storage

## Tech Stack

| Area | Technology |
|------|-----------|
| UI | SwiftUI |
| Backend | Firebase (Auth, Firestore, Storage) |
| Maps | MapKit |
| Real-time | Firestore listeners |

## Highlights

- Dual-role architecture with role-based access control
- Real-time order pipeline — customers see live status updates as admins process orders
- In-app wallet reduces dependency on external payment flows for demo/prototype use
